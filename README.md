# ResetPass
Скрипт для сброса пароля пользователя на стандартский 

## Содержание
- [Требования](#требования)
- [Использование](#использование)

### Требования
Запускать на локальной машине пользователя под администратором

### Использование
1. Ввести фамилию пользователя или логин
2. Проверить корректность данных, если допустили ошибку - нажмите любую клавишу и повторно введите фамилию, в противном случае нажмите ввод
